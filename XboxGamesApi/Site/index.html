About Xbox Game Browser

An example application built by Shawn Wildermuth of Wilder Minds LLC

This application was built using client-side JavaScript and a JSON-powered back-end. This site is an example of how to build and application that can be easily ported to the iPhone, iPad, Android, Windows Phone 8 and Windows 8 using PhoneGap and Adobe PhoneGap Build.

This site uses a variety of client-side libraries including:

  • jQuery
  • Twitter Bootstrap
  • KnockoutJS
  • AmplifyJS
  • Sammy.js

In addition, the site was initially created using Initializr to produce a HTML5 Boilerplate project with Twitter Bootstrap and other features.

The template was provided by BootSwatch and icons are from the Twitter Bootstrap project but originally supplied by GlyphIcons.
